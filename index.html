HTML

Linguagem de Marcação de Texto, usado para dar significado a estrutura do conteúdo do site.

CSS

Linguagem de Estilo usada para descrever a apresentação de um documento escrito em html.

GIT

Usado para versionamento de código e também para registrar o histórico de edições do arquivo.

JavaScript

Linguagem de programação front/back-end, usada com HTML e CSS para melhorar a dinâmica e interação com os usúarios.

Node Js

Software de código aberto, multiplataforma, que permite a execução de códigos JavaScript fora de um navegador web.
